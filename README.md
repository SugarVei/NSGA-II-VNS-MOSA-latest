# 多目标调度优化系统
# Multi-Objective Scheduling Optimization System

## 功能特性
- 三种元启发式算法: NSGA-II, VNS, MOSA
- 可视化: 3D Pareto前沿图、收敛曲线
- PyQt5 图形界面
- 支持手动输入和自动生成数据

## 安装依赖
```bash
pip install -r requirements.txt
```

## 运行程序
```bash
python main.py
```

## 项目结构
```
scheduling_optimizer/
├── main.py                 # 程序入口
├── models/                 # 数据模型
├── algorithms/             # 优化算法
├── ui/                     # 用户界面
├── visualization/          # 可视化
└── utils/                  # 工具函数
```
